This is a Java game which works like warcarft.
The GUI is written by JavaFX scene builder.

If you want to run the game, 
1.  Restart NetBeans(or other IDE) first
1a. Open New project, put all the files in src in the project
2.  Open A2.java
3.  Clean and Build project
4.  Run Project

There are three modes, which are display assignment, single play, multi play.
If you want to change the game mode, do step 1,2,3,4 again.
